Principle:
\begin{itemize}
    \item Constant font
    \item Multi-columns, allow spanning over a few nearby columns
	\begin{itemize}
	    \item customized column width
	\end{itemize}
    \item It depends on authors to fit their content into frames, rather
	than changing font size.
\end{itemize}
